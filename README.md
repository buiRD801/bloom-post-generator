# 如光 社群貼文產生器

## 快速開始

### 本機開發
```bash
npm install
npm run dev
```
開啟 http://localhost:5173

### 部署到 Vercel（推薦）
1. 把這個資料夾推到 GitHub
2. 前往 https://vercel.com → New Project → 匯入你的 repo
3. 設定維持預設 → Deploy
4. 取得網址，分享給夥伴即可

### 部署到 GitHub Pages
```bash
npm install
npm run build
# 把 dist/ 資料夾內容推到 gh-pages branch
```

### 手動部署（任何靜態伺服器）
```bash
npm install
npm run build
# dist/ 資料夾就是完整網站，上傳到任何伺服器即可
```
